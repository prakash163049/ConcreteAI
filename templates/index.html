How It Works

Project Workflow

From raw data to real-time ML predictions — a streamlined engineering pipeline built for accuracy.

Data Collection

UCI ML Repository concrete dataset with 1,030 real-world samples across 9 variables.

Data Cleaning

Handled missing values, removed duplicates, and normalized units for consistency.

Exploratory Data Analysis

Distribution plots, correlation heatmaps, and outlier detection using IQR method.

Feature Engineering

Scaled features, built interaction terms, and selected the 8 most impactful predictors.

Model Training

Compared 10+ algorithms with cross-validation; Extra Trees emerged as the best performer.

Extra Trees Regressor

Achieved 92.12% R² score. Serialized model via Pickle for production deployment.

Real-Time Prediction

Flask API serves instant predictions to this web interface in milliseconds.
